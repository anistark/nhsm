Node Hapi Swig Mongo
=====================

Stuffs Used :

#### Node

#### Hapi

#### Swig

#### Mongoose

#### Joi


Setup
```
$ git clone --depth=1 https://github.com/anistark/nhsm.git
$ cd nhsm
$ npm install
```

Start server:
```
$ node server
```

#### Production

Good to minify first:
```
npm install -g gulp
```
